# Student API

A simple FastAPI project for managing student records.

## Features

- Get all students
- Get a student by ID
- Add a new student
- Update an existing student
- Delete a student

## Technologies Used

- FastAPI
- SQLAlchemy
- Pydantic
- SQLite
